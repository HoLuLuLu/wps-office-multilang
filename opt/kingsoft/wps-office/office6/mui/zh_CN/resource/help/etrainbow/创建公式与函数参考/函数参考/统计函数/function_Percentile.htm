PERCENTILE

返回区域中数值的第 K 个百分点的值。可以使用此函数来建立接受阈值。例如，可以确定得分排名在第 90 个百分点之上的检测侯选人。

语法

PERCENTILE(array,k)

Array     为定义相对位置的数组或数据区域。

K     0 到 1 之间的百分点值，包含 0 和 1。

说明

  • 若 array 为空或其数据点超过 8,191 个，则 PERCENTILE 返回错误值 #NUM!。
  • 若 k 为非数字型，则 PERCENTILE 返回错误值 #VALUE!。
  • 若 k < 0 或 k > 1，则 PERCENTILE 返回错误值 #NUM!。
  • 若 k 不是 1/(n-1) 的倍数，则 PERCENTILE 使用插值法来确定第 k 个百分点的值。
