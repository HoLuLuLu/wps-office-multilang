VARPA

计算基于整个样本总体的方差。不仅数字，文本值和逻辑值（如 TRUE 和 FALSE）也将计算在内。

语法

VARPA(value1,value2,...)

Value1, value2,...     为对应于样本总体的 1 到 30 个参数。

说明

  • 函数 VARPA 假设参数即为样本总体。若数据代表的是总体的一个样本，则必须使用函数 VARA 来估计方差。
  • 包含 TRUE 的参数作为 1 计算。包含文本或 FALSE 的参数作为 0 计算。若在计算中不能包含文本值或逻辑值，请使用 VARP 工作表函数来代替。
