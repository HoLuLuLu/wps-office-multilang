EXPONDIST

返回指数分布。使用函数 EXPONDIST 可以建立事件之间的时间间隔模型，例如，在计算银行自动提款机支付一次现金所花费的时间时，可通过函数 EXPONDIST 来确定这一过程最长持续 1 分钟的发生概率。

语法

EXPONDIST(x,lambda,cumulative)

　

 X   函数的值。

 Lambda   参数值。

 Cumulative   一逻辑值，指定要提供的指数函数的形式。如果 cumulative 为 TRUE，函数 EXPONDIST 返回累积分布函数；如果 cumulative 为 FALSE，返回概率密度函数。

说明

  • 若 x 或 lambda 为非数值型，则 EXPONDIST 返回错误值 #VALUE!。
  • 若出现以下任一情况 ，则 EXPONDIST 返回错误值 #NUM!。

        1、x < 0

        2、lambda ≤ 0
