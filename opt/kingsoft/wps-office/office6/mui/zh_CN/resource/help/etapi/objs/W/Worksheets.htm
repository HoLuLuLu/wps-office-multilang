Worksheets 集合

请参阅  属性  方法  事件  特性 

Application

  Worksheets

    HPageBreaks;VPageBreaks;Worksheet

　

描述

指定工作簿中或活动工作簿中所有 Worksheet 对象的集合。每个 Worksheet 对象代表一张工作表。

　

说明

可用 Worksheets 属性返回 Worksheets 集合。下例显示活动工作簿所有工作表的个数。

Msgbox Worksheets.Count

可用 Add 方法新建一张工作表并将其添加到集合中。下例在活动工作簿的第二张工作表后面添加一张新的工作表。

Worksheets.Add Count:=1, After:=Sheets(2)

可用 Worksheets(index) 返回单个 Worksheet 对象，其中 index 为工作表编号或名称。下例隐藏活动工作簿中的工作表sheet2。

Worksheets("sheet2").Visible = False

Worksheet 对象也是 Sheets 集合的成员。Sheets 集合包含工作簿中所有的工作表。
