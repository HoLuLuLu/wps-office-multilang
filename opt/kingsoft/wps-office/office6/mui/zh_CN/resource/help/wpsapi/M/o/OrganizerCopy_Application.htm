描述

将指定的“自动图文集”词条、工具栏、样式或宏方案项从源文档或模板中复制到目标文档或模板上。


语法

expression.OrganizerCopy((Source), (Destination), (Name), (Object))

部  分 描  述
expression 必需。该表达式返回 Application 对象。
Source 必需，String 类型。为含有需要复制的条目的源文档或模板的名称。
Destination 必需，String 类型。为需要复制条目的目标文档或模板的名称。
Name 必需，String 类型。为需要复制的“自动图文集”词条、工具栏、样式或宏的名称。
Object 必需，wpsOrganizerObject 常量。需要复制的项类型。
