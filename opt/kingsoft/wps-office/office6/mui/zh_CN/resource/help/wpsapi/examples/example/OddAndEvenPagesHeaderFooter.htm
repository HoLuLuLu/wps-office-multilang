本示例为 Document1 的奇偶页创建不同的页眉和页脚。

Set myWps = Documents("Document1")
myWps.PageSetup.OddAndEvenPagesHeaderFooter = True
With myWps.Sections(1)
    .Headers(wpsHeaderFooterPrimary).Range _
    .InsertAfter "Odd Header"
    .Headers(wpsHeaderFooterEvenPages).Range _
    .InsertAfter "Even Header"
End With
