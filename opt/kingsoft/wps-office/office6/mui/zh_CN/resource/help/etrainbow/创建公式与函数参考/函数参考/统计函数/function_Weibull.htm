WEIBULL

返回韦伯分布。使用此函数可以进行可靠性分析，比如计算设备的平均故障时间。

语法

WEIBULL(x,alpha,beta,cumulative)

X     参数值。

Alpha     分布参数。

Beta     分布参数。

Cumulative     确定函数的形式。

说明

  • 若 alpha ≤ 0 或 beta ≤ 0，则 WEIBULL 返回错误值 #NUM!。
  • 若 x < 0，则 WEIBULL 返回错误值 #NUM!。
  • 若 x、alpha 或 beta 为非数值型，则 WEIBULL 返回错误值 #VALUE!。
