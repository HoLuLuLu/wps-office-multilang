描述

滚动文档窗口，以便在文档窗口显示指定的区域或图形。


语法

expression.ScrollIntoView((Obj), [Start])

部  分 描  述
expression 必需。该表达式返回 View 对象。
Obj 必需，Object 类型。可以是 Selection、Range 或 Shape 对象。
Start 可选，Boolean 类型。如果此值为 True，该区域或图形的左上角将出现在文档窗口的左上角。如果此值为 False，该区域或图形的右下角将出现在文档窗口的右下角。默认值为 True。


说明

如果该区域或图形大于文档窗口，Start 参数可以指定该区域或图形的显示部分或获得初始焦点的部分。此方法无法在大纲视图中使用。
