MROUND 

　

返回参数按指定基数舍入后的数值。

语法

MROUND (number, multiple)

　

Number   要进行四舍五入的值。

　

Multiple   要将数值 number 进行四舍五入的基数。

　

说明

　

如果数值 number 除以基数的余数大于或等于基数的一半，则函数 MROUND 向远离零的方向舍入。
