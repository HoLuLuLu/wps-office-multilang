Comments 对象

请参阅  属性  方法  事件  特性 

Range

  Comments(Comment)

　

描述

代表一条单元格批注。Comment 对象是 Comments 集合的成员。

　

说明

1.可用 AddComment 方法为一个区域添加批注。下例为第一张工作表的 D5 单元格添加批注。

With Worksheets(1).Range("D5").AddComment
    .Visible = False
    .Text "reviewed on " & Date
End With

2.可用 Comments 属性返回 Comments 集合。下例隐藏第一张工作表中的所有批注。

Set dd = Worksheets(1).Comments
For Each c In dd
    c.Visible = False
Next

3.使用 Comments(index) 可返回 Comments 集合中的单条批注，其中 index 为批注号。下例隐藏第一张工作表中的批注2。

Worksheets(1).Comments(2).Visible = False
