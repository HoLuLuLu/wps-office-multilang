本示例在活动文档中设置“标题 1”样式后接“标题 2”样式。

ActiveDocument.Styles(wpsStyleHeading1).NextParagraphStyle = _
ActiveDocument.Styles(wpsStyleHeading2)

　

本示例新建一篇文档并添加一个名为“MyStyle”的段落样式。该新样式根据“正文”样式创建，后接“标题 3”样式，左缩进量为 1 英寸（72 磅），设为加粗格式。

Set myDoc = Documents.Add
Set myStyle = myDoc.Styles.Add(Name:= "MyStyle")
With myStyle
    .BaseStyle = wpsStyleNormal
    .NextParagraphStyle = wpsStyleHeading3
    .ParagraphFormat.LeftIndent = 72
    .Font.Bold = True
End With
