本示例在显示文档的原始版本和最终版本之间切换。运行该示例前，假定活动窗口中的文档包含多个审阅者所作的修订，并且修订显示于气球中。

With ActiveWindow.View
    If .RevisionsMode = wpsBalloonRevisions Then
        If .RevisionsView = wpsRevisionsViewFinal Then
            .RevisionsView = wpsRevisionsViewOriginal
        Else
            .RevisionsView = wpsRevisionsViewFinal
        End If
    End If
End With
