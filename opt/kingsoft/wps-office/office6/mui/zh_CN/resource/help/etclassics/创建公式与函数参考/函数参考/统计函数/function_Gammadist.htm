GAMMADIST

返回伽玛分布。可以使用此函数来研究具有偏态分布的变量。伽玛分布通常用于排队分析。

语法

GAMMADIST(x,alpha,beta,cumulative)

X     用来计算分布的值。

Alpha     分布参数。

Beta     分布参数。若 beta = 1，则 GAMMADIST 返回标准伽玛分布。

Cumulative     一逻辑值，决定函数的形式。若 cumulative 为 TRUE，则 GAMMADIST 返回累积分布函数。若为 FALSE，则返回概率密度函数。

说明

  • 若 x、alpha 或 beta 为非数值型，则 GAMMADIST 返回错误值 #VALUE!。
  • 若 x < 0，则 GAMMADIST 返回错误值 #NUM!。
  • 若 alpha ≤ 0 或 beta ≤ 0，则 GAMMADIST 返回错误值 #NUM!。
