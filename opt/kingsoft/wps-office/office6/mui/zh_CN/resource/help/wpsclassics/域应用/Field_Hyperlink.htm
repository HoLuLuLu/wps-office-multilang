Hyperlink

超链接是一个“热点”，可以从一个位置跳转至其他位置。这种位置可包括硬盘或局域网上的其他文件（如 金山文档或金山工作表）、Internet 地址（如 http://www.kingsoft.com/）、书签或幻灯片。此域包括提示文字，一般为蓝色并带有下划线，用户可单击此处跳转至指定位置。

  • 通过普通方式建立超链接
    请按照如下步骤：

    1. 选中需要添加超链接的文字。
      2. 在“常用”工具栏上单击“插入超链接”，如下图所示：
    2. 单击“确定”。

  • 通过 Hyperlink 域方式建立超链接
    { Hyperlink "FileName" [ Switches ] }

     说明

    “FileName”

    		 要跳转到的目标的位置。如果其中包含较长的带空格文件名，请用引号引住。以双反斜杠替代单反斜杠指定路径。例如：“C:\\My Documents\\kingsoft.wps”
    对于 Internet 地址，则包含与 URL 一样的协议和语法，例如：“http://www.kingsoft.com/news/index.shtml”。

     开关

    \l 指定此超链接跳转到的文件中的位置，如书签等。
    \o 指定超链接的屏幕提示文字。
