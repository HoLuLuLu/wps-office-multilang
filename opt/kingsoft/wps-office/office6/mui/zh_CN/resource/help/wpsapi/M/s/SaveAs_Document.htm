描述

用一个新的文件名或格式来保存指定文档。此方法的参数与“文件”菜单的“另存为”对话框中的选项相对应。


语法

expression.SaveAs([FileName], [FileFormat], [LockComments], [Password], [AddToRecentFiles], [WritePassword], [ReadOnlyRecommended], [EmbedTrueTypeFonts], [SaveNativePictureFormat], [SaveFormsData], [SaveAsAOCELetter], [Encoding], [InsertLineBreaks], [AllowSubstitutions], [LineEnding], [AddBiDiMarks])

部  分 描  述
expression 必需。该表达式返回 Document 对象。
FileName 可选，String 类型。文档名称。默认值是当前文件夹和文件名。如果从未保存过此文档，则将使用默认的文件名（例如，文档 1.wps）。如果已有文件名指定的文档，则覆盖此文档，而且在覆盖前不提醒用户。
FileFormat 可选，String 类型。文档保存的格式。可以是任意 WpsSaveFormat 常量。若要将文档保存为另一种格式，请使用 FileConverter 对象的 SaveFormat 属性指定相应的值。
LockComments 可选，Boolean 类型。如果为 True，则锁定文档备注。默认值是 False。
Password 可选，String 类型。打开文档的密码。
AddToRecentFiles 可选，Boolean 类型。如果为 True，则将文档添加至“文件”菜单中最近使用的文件列表中。默认值为 True。
WritePassword 可选，String 类型。保存对文档的修改所需的密码。
ReadOnlyRecommended 可选，Boolean 类型。如果为 True，在每次打开文档时，WPS 将建议用户采用只读方式。默认值为 False。
EmbedTrueTypeFonts 可选，Boolean 类型。如果为 True，则将文档与 TrueType 字体一起保存。如果忽略，则 EmbedTrueTypeFonts 参数将假定为 EmbedTrueTypeFonts 属性的值。
SaveNativePictureFormat 可选，Boolean 类型。如果为 True，则对于从其他系统平台（例如 Macintosh）导入的图形，将仅保存其 Windows 版本。
SaveFormsData 可选，SaveFormsData 类型。如果为 True，则将用户在窗体中输入的数据保存为数据记录。
SaveAsAOCELetter 可选，Boolean 类型。如果文档包含附加的邮件，当此属性值为 True 时，将文档存为 AOCE 信函（同时保存邮件）。
Encoding 可选，Long 类型。将文档保存为编码文本文件时，所用的代码页或字符集。默认值为系统代码页。
InsertLineBreaks 可选，Boolean 类型。在将文档保存为文本文件时，如果为 True，在每一行文本后插入换行符。
AllowSubstitutions 可选，Boolean 类型。在将文档保存为文本文件时，如果为 True，使 WPS 可以将一些符号替换为相似的文本。例如，将版权符号显示为 (c)。默认值是 False。
LineEnding 可选，Long 类型。Word 在文档中标记换行符和分段符的方式被保存为文本文件。可以是任意 wpsLineEndingType 常量。
AddBiDiMarks 可选，Long 类型。如果为 True，则为输出文件添加控制符，以保留原始文档中文本的双向版式。
