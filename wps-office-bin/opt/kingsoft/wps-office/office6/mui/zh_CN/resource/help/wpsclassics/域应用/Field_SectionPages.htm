SectionPages

{ SectionPages }

说明

SectionPages 域是插入一节中的总页数。使用该域时，必须将第一节之后每一节的页从 1 开始重新编号。

开关

\* 设置数字的开关格式，如：图表1可提供的格式有：图表一、图表①、图表壹、图表甲等等

示例

例如：要在已分节（节：文档的一部分，可在其中设置某些页面格式选项。若要更改例如行编号、列数或页眉和页脚等属性，请创建一个新的节。）的文档的每一页上打印如“第 3 页 共 9 页”之类的文字，可在页眉或页脚中插入下列域和文字：

  1. 将光标置于要插入域的文档位置或进入页眉或页脚区域，键入文字。
  2. 按 Ctrl+F9 ，键入 Page 和 SectionPages 域，如下：
    域代码 域结果
    第{ PAGE } 共 { SECTIONPAGES } 页 第 3 页 共 9 页
