下面的示例将变量设置为工作表“sheet1”上的筛选区域中第一列的筛选 Criteria1 属性值。

Dim c2 As String

With ThisWorkbook.Worksheets("sheet1")

    If .AutoFilterMode Then

        With .AutoFilter.Filters.Item(1)

            If .On And .Operator Then

                 c2 = .Criteria2

            Else

                c2 = "Not set"

            End If

        End With

    End If

End With
