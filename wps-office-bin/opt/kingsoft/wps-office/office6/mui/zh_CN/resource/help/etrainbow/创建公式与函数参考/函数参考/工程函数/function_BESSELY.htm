BESSELY

返回 Bessel 函数值，也称为 Weber 函数或 Neumann 函数。

语法

BESSELY(X, N)

X  参数值：用于测试函数的值。

N  函数的阶数：函数的阶数

　

说明

  • 如果 x 是非数值型，则 BESSELY 返回 #VALUE! 错误值。
  • 如果 n 是非数值型，则 BESSELY 返回 #VALUE! 错误值。
  • 如果 n < 0，则 BESSELY 返回 #NUM! 错误值。
