FISHER

返回点 x 的 Fisher 变换。该变换生成一个正态分布而非偏斜的函数。使用此函数可以完成相关系数的假设检验。

语法

FISHER(x)

X    要对其进行变换的数值。

说明

  • 若 x 为非数值型，则 FISHER 返回错误值 #VALUE!。
  • 若 x ≤ -1 或 x ≥ 1，则 FISHER 返回错误值 #NUM!。
