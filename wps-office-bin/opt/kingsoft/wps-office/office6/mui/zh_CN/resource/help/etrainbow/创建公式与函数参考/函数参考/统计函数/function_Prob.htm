PROB

返回区域中的数值落在指定区间内的对应概率。如果没有给出上限 upper_limit，则返回区间 x_range 内的值等于下限 lower_limit 的概率。

语法

PROB(x_range,prob_range,lower_limit,upper_limit)

x_range 具有各自相应概率值的 x 数值区域。

prob_range 与 x_range 中的各个值相对应的一组概率值。

lower_limit 用于函数计算概率的数值下界。

upper_limit 用于函数计算概率的可选数值上界。

说明

  • 如果 prob_range 中的任意值 ≤ 0 或 > 1，函数 PROB 返回错误值 #NUM!。
  • 如果 prob_range 中所有值之和大于1，函数 PROB 返回错误值 #NUM!。
  • 如果省略 upper_limit，函数 PROB 返回值等于 lower_limit 时的概率。
  • 如果 x_range 和 prob_range 中的数据点对应的个数不同，函数 PROB 返回错误值 #N/A。
