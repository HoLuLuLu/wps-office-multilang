Application

  Chart；Recentfiles；Windows；Workbooks；Worksheet

　

描述

代表整个金山表格应用程序。Application 对象包含：

  • 应用程序范围内的设置和选项。
  • 返回顶级对象的方法，例如 ActiveCell 和 ActiveSheet 等。

　

说明

可用 Application 属性返回 Application 对象。下例将 Windows 属性应用于 Application 对象，并激活工作簿。

Application.Windows("book1.et").Activate

许多返回常用用户界面对象的属性和方法，例如活动单元格（ActiveCell 属性），可不加 Application 对象识别符而直接使用。

例如，可用 ActiveCell.Font.Bold = True 代替 Application.ActiveCell.Font.Bold = True。
