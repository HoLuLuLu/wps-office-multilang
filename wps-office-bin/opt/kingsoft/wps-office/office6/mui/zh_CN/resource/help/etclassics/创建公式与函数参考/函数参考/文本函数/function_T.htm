T

返回 value 引用的文本。

语法

T(value)

Value     需要进行测试的数值。

说明

  • 若值是文本或引用文本，T 返回值。如果值不引用文本，T 返回空文本 ("")。
  • 通常不需在公式中使用函数 T，因为 WPS 表格可以自动按需要转换数值的类型，该函数用于与其他电子表格程序兼容。

示例

如果您将示例复制到空白工作表中，可能会更易于理解该示例。

A
1 数据
2 45
3 开花
4 TRUE
5 公式 说明（结果）
6 =T(A2) 因为第一个 value 是数字，所以返回空文本 ( )
7 =T(A3) 因为第二个 value 是文本，所以返回文本 (开花)
8 =T(A4) 因为第三个 value 是逻辑值，所以返回空文本 ( )
