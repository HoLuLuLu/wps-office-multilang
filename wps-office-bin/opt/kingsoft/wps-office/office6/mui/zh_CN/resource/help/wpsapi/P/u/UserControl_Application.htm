描述

如果该属性值为 True，则文档或应用程序是由用户创建或打开的。如果该属性值为 False，则文档或应用程序是由其他 WPS 应用程序用 Open 方法以及 CreateObject 或 GetObject 方法创建或打开的。


返回 String 类型 ，可读写。


语法

expression.UserControl

部  分 描  述
expression 必需。该表达式返回 Application 对象。


说明

如果 WPS 对用户为可见的，或用户在 WPS 代码模块内调用 Application 或 Document 对象中的 UserControl 属性，则该属性返回 True。
