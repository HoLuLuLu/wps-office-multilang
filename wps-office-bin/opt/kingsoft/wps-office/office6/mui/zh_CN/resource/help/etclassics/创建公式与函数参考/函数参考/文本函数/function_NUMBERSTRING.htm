NUMBERSTRING

将数字转换为中文字符串。

语法

NUMBERSTRING(VALUE,TYPE)

VALUE 数值：被转换为中文字符串的数值。

TYPE 选项，操作选项。有3种，分别为1，2，3。

说明：

  • 此函数仅支持正整数，带小数时会截取小数取整。
Type 　
参数 说明
1 返回数值的中文计数说明
2 返回数值的大写中文计数说明
3 返回数值中文文字说明

示例

如果您将示例复制到空白工作表中，可能会更易于理解该示例。

A B
1 数值 公式（结果）
2 12345 =NUMBERSTRING(A2,1)（一万二千三百四十五）
3 12345 =NUMBERSTRING(A2,2)（壹万贰仟叁佰肆拾伍）
4 12345 =NUMBERSTRING(A2,3)（一二三四五）
