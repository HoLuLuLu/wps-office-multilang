FORECAST

根据已有的数值计算或预测未来值。此预测值为基于给定的 x 值推导出的 y 值。已知的数值为已有的 x 值和 y 值，再利用线性回归对新值进行预测。可以使用该函数对未来销售额、库存需求或消费趋势进行预测。

语法

FORECAST(x,known_y's,known_x's)

x 为需要进行预测的数据点。

known_y's 为因变量数组或数据区域。

known_x's 为自变量数组或数据区域。

说明

  • 如果 x 为非数值型，函数 FORECAST 返回错误值 #VALUE!。
  • 如果 known_y's 和 known_x's 为空或含有不同个数的数据点，函数 FORECAST 返回错误值 #N/A。
  • 如果 known_x's 的方差为零，函数 FORECAST 返回错误值 #DIV/0!。
