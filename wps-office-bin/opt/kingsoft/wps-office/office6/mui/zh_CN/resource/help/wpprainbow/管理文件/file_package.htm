文件打包

在实际工作生活过程中，用户常常需要在演示文稿中添加硬盘中的音乐、视频等文件，如果这些音乐、视频文件在硬盘中更改了路径，或者该演示文稿存储在另一个硬件设备中，当再次打开这个演示文稿，由于关联路径的改变，将无法调用这些音乐和视频文件。

WPS演示新增文件打包功能，将演示文稿和文稿内的所有音频或视频文件存储在同一个文件夹或压缩文件内，方便携带储存。

请按以下步骤进行操作：

  1. 在演示文稿中插入所需的音频、视频文件，并保存。
  2. 在“插入”选项卡上，单击“文件打包”，从下拉菜单中选取“打包成文件”或“将演示文稿打包成压缩文件”。
  3. 弹出“演示文件打包”对话框，单击“浏览”，选取文件保存位置。
  4. 键入文件夹名称或压缩文件名。
  5. 单击“确定”，在指定的路径下产生一个文件夹或压缩文件。
