本示例检查活动文档第三段的文本格式是否设置为全部字母大写。

If ActiveDocument.Paragraphs(3).Range.Font.AllCaps = True Then
    MsgBox "第三段的所有文字都是大写的。"
Else
    MsgBox "第三段存在不是大写的文字。"
End if

　

本示例将选定文本的字体格式设置为全部字母大写。

If Selection.Type = wpsSelectionNormal Then
    Selection.Font.AllCaps = True
Else
    MsgBox "请选择一些文字以执行操作。"
End If
