文章需要题目，同样，图片、表格等相对独立的文档元素也需要“标题”，大量使用时甚至需要将它们编号以备查阅，这些“标题”便是题注。示例如下图所示：

WPS文字提供插入的题注功能能对使用同一标签的题注编号自动递增。

  • 通过普通方式添加题注

    请按照如下步骤：

    1. 选中需要添加题注的文档元素（通常是表格、图片或OLE对象）。
      2. 在“插入”菜单下，单击“引用”，在下级菜单中选择“题注”，如下图所示：
    2. 单击“确定”。
  • 通过 SEQ 域方式添加题注
    { SEQ Identifier [Bookmark ] [Switches ] }
    对文档中的章节、表格、图表和其他项目按顺序编号。如果添加、删除或移动一个项目及其相应的 SEQ 域，则可以更新文档中余下的 SEQ 域以得到新的序号。

     说明

    标识符

    		为要编号的一系列项目指定的名称。例如，一系列表格的名称可是“tables”或“表_1”。
    书签 加入书签名来引用文档中其他位置的项目。例如，要引用一个图解，可用书签“pic2”标注给该图解编号的 SEQ 域，然后用 { SEQ 图示 Pic2 } 插入对其的引用。

     开关

    \h 隐藏域结果（ 域结果：当执行域指令时，在文档中插入的文字或图形。在打印文档或隐藏域代码时，将以域结果替换域代码。）。
    \n 为指定项目插入下一个序号。该开关是默认的。
    \r n 将序号重置为指定的值 n。例如，{ SEQ figure \r 3 } 对图表从 3 开始编号。
    \s

    在“s”后的标题级别处重新设置序号。例如，{ SEQ figure \s 2 } 从标题 2样式处开始编号。

     示例

    如果不使用样式来为文件标题设置格式，可以通过在每章开始处插入 SEQ 域来为章节标题编号：　
    域代码 域结果
    Chapter { SEQ chapter } Chapter 3

 提示

  • 如系统提供的标签不能满足需要，请单击“新建标签”自定义。
