BETAINV

返回指定的 beta 分布累积分布函数的反函数值。即，如果 probability = BETADIST(x,...)，则 BETAINV(probability,...) = x。beta 分布函数可用于项目设计，在给定期望的完成时间和变化参数后，模拟可能的完成时间。

语法

BETAINV(probability,alpha,beta,A,B)

Probability     Beta 分布的概率值。

Alpha     分布参数。

Beta     分布参数。

A     数值 x 所属区间的可选下界。

B     数值 x 所属区间的可选上界。

说明

  • 若任意参数为非数值型，则 BETAINV 返回错误值 #VALVE!。
  • 若 alpha ≤ 0 或 beta ≤ 0，则 BETAINV 返回错误值 #NUM!。
  • 若 probability ≤ 0 或 probability > 1，则 BETAINV 返回错误值 #NUM!。
  • 若省略 A 或 B 值，则 BETAINV 使用标准的累积 beta 分布，即 A = 0，B = 1。

若已给定概率值，则 BETAINV 使用 BETADIST(x, alpha, beta, A, B) = probability 求解数值 x。因此，BETAINV 的精度取决于 BETADIST 的精度。BETAINV 使用迭代搜索技术。若搜索在 100 次迭代之后没有收敛，则函数返回错误值 #N/A。
