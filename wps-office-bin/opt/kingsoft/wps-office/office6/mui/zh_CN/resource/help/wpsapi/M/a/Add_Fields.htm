描述

将 Field 对象添加到 Fields 集合。在指定区域中返回 Field 对象。


语法

expression.Add((Range), [Type], [Text], [PreserveFormatting])

部  分 描  述
expression 必需。该表达式返回 Fields 对象集合。
Range 必需，Range 对象。需要添加域的区域。如果该区域未折叠，那么域将替换此区域。
Type 可选，WpsFieldType 常量。有关有效的常量列表的详细信息，请参阅“对象浏览器”。默认值为 wpsFieldEmpty。
Text 可选，Variant 类型。域所需的其他文本。例如，如果要给域指定一个开关，可将其添加至此处。
PreserveFormatting 可选，Variant 类型。如果为 True，则更新时保留域所应用的格式。


说明

Fields 集合的 Add 方法插入诸如 wpsFieldOCX 和 wpsFieldFormCheckBox 之类的域。而对于 FormFields 集合必须使用诸如 AddOLEControl 和 Add 之类的特定方法。
